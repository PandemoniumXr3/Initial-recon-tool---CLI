# LSR - Local System Recon

LSR is a simple Python CLI tool for basic local system reconnaissance.

## What it does
- Shows the current user
- Shows group memberships
- Shows the hostname
- Checks sudo privileges

## How to use
Run the script from the project directory:

```bash
python3 lsr.py
